# Screenshots Directory

Questa directory contiene gli screenshot per il manuale utente.

## Screenshot Richiesti

Per completare il manuale utente, aggiungi i seguenti screenshot:

### Homepage e Lobby
- `homepage.png` - Schermata iniziale con campo nickname
- `lobby-host.png` - Vista della lobby per l'host
- `lobby-player.png` - Vista della lobby per un giocatore normale

### Giocatore Normale
- `player-normal-word.png` - Schermata con la parola segreta visibile
- `player-normal-victory.png` - Schermata di vittoria per i giocatori normali

### Impostore
- `impostor-screen.png` - Schermata dell'impostore con suggerimento
- `impostor-victory.png` - Schermata di vittoria per l'impostore

### Chat Vocale
- `voice-chat-panel.png` - Pannello della chat vocale disconnesso
- `voice-chat-active.png` - Pannello della chat vocale attivo con partecipanti

## Come Aggiungere Screenshot

1. Avvia l'applicazione in locale: `dotnet run`
2. Gioca una partita completa
3. Cattura gli screenshot richiesti
4. Salva i file in questa directory con i nomi indicati sopra
5. Gli screenshot saranno automaticamente referenziati nel `MANUALE_UTENTE.md`
